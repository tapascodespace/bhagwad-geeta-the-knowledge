📘 Bhagavad Gita — The Knowledge
A modern spiritual companion built for Gen Z and Gen Alpha.

🌟 Overview
Bhagavad Gita — The Knowledge is a clean, minimal, distraction‑free app that brings the timeless wisdom of the Gita to a new generation of digital‑native learners.
Designed for clarity, focus, and daily reflection, the app reimagines ancient philosophy through a modern UI and smooth reading experience.

This project is fully open‑source and built with simplicity at its core.

✨ Features
Complete Bhagavad Gita with chapter‑wise navigation

Minimal, modern UI crafted for Gen Z & Gen Alpha

Offline access for uninterrupted reading

Fast, intuitive navigation

Clean typography for long‑form spiritual reading

Lightweight & distraction‑free
